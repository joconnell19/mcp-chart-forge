OpenClaw Blocked

April 4 | Anthropic cuts off third-party tool access from subscriptions

What happened

Pro/Max subscriptions no longer work with third-party tools. Even mentioning "OpenClaw" in a system prompt triggers a 400 error.

Why

Third-party tools bypass prompt caching, consuming significantly more compute. ~135k OpenClaw instances affected, some facing up to 50x cost increases.

Anthropic blocked all third-party tool access from subscriptions on April 4. But this week it went further - even using the official Claude Code CLI with "OpenClaw" mentioned in your system prompt returns a 400 error. They're scanning prompt content now, not just blocking OAuth. OpenClaw creator Steinberger, who left for OpenAI in February, said "funny how timings match up, first they copy features into their closed harness, then they lock out open source."

Claude Mythos

Anthropic's most capable model ever. Found thousands of zero-days across every major OS and browser, including a 27-year-old OpenBSD vulnerability. Too powerful to release publicly.

Project Glasswing

Gives defenders early access to Mythos. Launch partners include AWS, Apple, Google, Microsoft, Nvidia, CrowdStrike, and more.

Announced yesterday. Mythos is Anthropic's most powerful model - it was leaked in the source code incident and they've now officially acknowledged it. It's exceptionally good at finding security vulnerabilities - thousands of zero-days including a 27-year-old bug in OpenBSD. They consider it too dangerous to release publicly, so Project Glasswing gives major tech companies early access for defensive security work. The idea is to give the good guys a head start before these capabilities become widely available.
